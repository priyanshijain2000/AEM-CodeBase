Disclosing Personal Test-taker information

ETS Customer Service Representatives can give out personal information only to the test taker, as required by privacy laws. If you are not the test taker, our representatives can only help you with general information, such as score report mailing dates, yes or no answers to general questions and payment processing for outstanding balances.
